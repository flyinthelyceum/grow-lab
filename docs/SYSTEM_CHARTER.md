# System Charter

## Project Name
Living Light System

## Vision

Create an open-air living plant sculpture where biological life and engineered systems coexist visibly and elegantly.

Plants appear to grow in a magical environment, while careful observation reveals an integrated technological architecture supporting them.

The system explores the intersection of:

- plant biology
- lighting engineering
- environmental sensing
- data logging
- computational control
- sculptural design

The final installation should feel both **alive and intelligent**.

## Goals

1. Grow healthy plants under controlled indoor conditions.
2. Instrument the biological environment with sensors.
3. Log environmental data over time.
4. Provide controllable lighting and irrigation.
5. Expose the technical systems aesthetically rather than hiding them.

## Design Philosophy

Technology should not dominate the organism.

The system should support plant growth while remaining visually elegant and conceptually legible.

## Design Principles

### 1. Visible Systems

Infrastructure should be legible. Lighting bars, sensors, tubing, and electronics may be visible but must be arranged intentionally.

### 2. Calm Complexity

The system may be complex internally but should appear calm and simple externally.

### 3. Biological Priority

Engineering decisions should support plant health. Technology should not dominate the organism.

### 4. Modularity

Each subsystem (lighting, irrigation, sensors, compute, structural frame) is independently replaceable without redesigning the entire system.

### 5. Data Transparency

Environmental data is a core component of the installation. Sensor measurements, system states, and control actions are continuously logged and made visible.

### 6. Iterative Construction

The project evolves through prototypes. Bench systems precede sculptural systems.

## Roadmap

### Phase 0 — Bench Prototype (V0)

Objective: prove core subsystems work together.

Focus: lighting, irrigation, sensor logging, basic plant growth.

Components:

- LED strip lighting system
- dimmable driver
- single plant bin
- drip irrigation + reservoir
- airflow fan
- temp/humidity sensor
- EC + pH monitoring
- soil moisture sensing
- Pi camera (timelapse)
- Raspberry Pi logging system

Deliverables:

- working light system
- working irrigation system
- sensor data logging
- stable plant growth

### Phase 1 — Sculptural System (V1)

Objective: transform prototype into aesthetic object.

- structural frame
- elegant reservoir integration
- hidden cable routing
- integrated lighting geometry
- Pi dashboard UI
- lighting automation

### Phase 2 — Environmental Intelligence (V2)

- humidity control
- automated irrigation logic
- reservoir level sensing
- plant growth feedback loops

### Phase 3 — Autonomous Ecology (V3)

Future research phase.

- machine learning plant optimization
- adaptive lighting spectrum
- multi-plant network
- distributed sensor mesh
